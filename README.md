SDL_3D Engine
=============
A simple 3D razterizer engine prototype made from scratch with SDL

Requirement:

SDL2

Don't forget to download the submodules :
git submodules init
git submodules update

Load object (example "./project ressources/TinyTina.obj")

Rotate :

x : q|d

y : s|z

z : a|e

Camera : 

y : up|down

x : left|right

z : keypad+|keypad-

View (wireframe|dot):

Toggle view : v
